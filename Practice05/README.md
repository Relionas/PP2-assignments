# Practice 5 – Python Regular Expressions (RegEx)

## Overview

This project demonstrates key Python programming concepts related to **Regular Expressions (RegEx)** and text processing.

The practice focuses on extracting structured information from raw text using pattern matching techniques.

### Topics Covered

- Basic RegEx exercises
- Pattern matching examples
- Receipt parsing using raw text data
- Structured JSON output generation

## Description

In this practice, Python's `re` module is used to work with **regular expressions** for finding and extracting patterns from text.

The project includes examples of how to:

- Search for patterns in text
- Extract specific data (such as product names, prices, dates)
- Process unstructured receipt text
- Convert extracted information into structured **JSON format**

## Technologies Used

- Python 3
- `re` (Regular Expressions module)
- `json` module

## Learning Objectives

By completing this practice you will learn how to:

- Use regular expressions in Python
- Extract useful information from raw text
- Work with pattern matching
- Convert parsed data into structured JSON format

## Example Use Case

One of the tasks in this project includes parsing a **store receipt** and extracting:

- Product names
- Prices
- Purchase date and time
- Payment method
- Total amount

The extracted data is then saved in a **JSON structure** for easier processing and storage.
